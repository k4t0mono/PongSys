Para o sistema funcionar corretamente deverá ser usado um usuario com todos os
privilégios necessários para fazer modificações no banco de dados. O nome do
usuário, a senha e o nome do banco de dados deverá ser configurado no arquivo
"persistence/conexao.php" antes de poder usar o sistema.

O arquivo principal do sistema é "view/index.php", o arquivo "index.php" na raiz
do projeto server para redirecionar para a pasta "view" onde contém os arquivos
do sistema.
